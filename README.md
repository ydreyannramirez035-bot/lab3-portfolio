# Laravel Portfolio Application

## Project Description
This is a simple Laravel portfolio web application created for Laboratory 3.  
It demonstrates routing, controllers, and Blade templates.

## Pages
- Home Page
- About Page
- Contact Page

## How to Run

1. Clone the repository
2. Run `composer install`
3. Run `php artisan serve`
4. Open http://127.0.0.1:8000
