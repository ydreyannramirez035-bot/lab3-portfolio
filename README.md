# Laravel Portfolio Application

## Project Description
This project is a simple portfolio website built using Laravel.  
It demonstrates the use of routes, controllers, and Blade templates to create multiple pages and navigate between them.

## How to Run the Project Locally
### ðŸ“¥ 1. Clone the Repository
Open Command Prompt or Terminal and run:
git clone https://github.com/ydreyannramirez035-bot/lab3-portfolio.git
Then go inside the project folder:
cd lab3-portfolio

---

### 2. Install Dependencies
Install required Laravel packages:
composer install

---

### 3. Create Environment File
Copy the example environment file:
copy .env.example .env
Then generate the application key:
php artisan key:generate

---

### 4. Run the Development Server
Start the Laravel local server:
php artisan serve
You should see something like:
Starting Laravel development server: http://127.0.0.1:8000

---

### 5. Open in Browser
Open your browser and visit:
http://127.0.0.1:8000
